# NestJS API Project

This is the NestJS API project within the monorepo. It serves as the backend for the application, providing necessary APIs for the Next.js frontend.

## Getting Started

To get started with the NestJS API, follow these steps:

1. **Install Dependencies**: Run the following command to install the required dependencies:
   ```
   npm install
   ```

2. **Run the Application**: Use the following command to start the NestJS application:
   ```
   npm run start
   ```

3. **Access the API**: Once the application is running, you can access the API at `http://localhost:3000`.

## Project Structure

- `src/main.ts`: The entry point of the application.
- `src/app.module.ts`: The root module of the application.
- `src/controllers/app.controller.ts`: Contains the controller for handling HTTP requests.
- `src/services/app.service.ts`: Contains the business logic for the application.

## Scripts

- `start`: Starts the application.
- `build`: Compiles the application.
- `test`: Runs the tests.
